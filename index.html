iOS Web Clip Profile Generator

Create a configuration profile (.mobileconfig) to add a website shortcut to your iOS home screen.

How to Use

  1. Fill details (Label, URL, Icon required).
  2. Optionally fill other fields & choose Full Screen.
  3. (Optional) Save configuration for later use.
  4. Click "Generate Profile".
  5. A `.mobileconfig` file will download.
  6. Transfer to iOS (Email, AirDrop, Files).
  7. Open on iOS device & go to Settings > General > VPN & Device Management.
  8. Tap profile, review (Consent Text), and Install (expect warnings).
  9. Shortcut appears on Home Screen.
