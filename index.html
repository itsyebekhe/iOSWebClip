iOS Web Clip Profile Generator

Create a configuration profile (.mobileconfig) to add a website shortcut to your iOS home screen.

How to Use

  1. Fill in the details above.
  2. Choose an icon image (PNG format, ideally square like 180x180px).
  3. Decide if you want Full Screen mode.
  4. Click "Generate Profile".
  5. A `.mobileconfig` file will be downloaded.
  6. Transfer this file to your iOS device (e.g., via Email, AirDrop, iCloud Drive).
  7. Open the file on your iOS device.
  8. Go to Settings > General > VPN & Device Management (or Profiles & Device Management).
  9. Tap on the downloaded profile and follow the prompts to install it. You will see warnings – proceed if you trust this generated file.
  10. The website shortcut will appear on your home screen.
